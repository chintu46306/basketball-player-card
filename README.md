## Installation

To get started with this project, follow the steps below:

### 1. Clone the repository:

 ```bash
   git clone https://github.com/your-username/basketball-player-card.git

```

### 2. Install the dependencies:

```
npm install


```

### 3. Start the application

```
npm start

```
